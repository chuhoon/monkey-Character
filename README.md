### 느낀점

1. z-index 10씩 주는 것이 추후에 헷갈리지 않는다.
2. border의 두께 box-sizing으로 감싸줘야 px이 달라지지 않는다.
3. 마진 병합현상 마진이 큰쪽으로 병합되어 2중으로 margin 값이 적용되지 않는다.
